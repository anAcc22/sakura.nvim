# Installation

## [lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{
    "anAcc22/sakura.nvim"
}
```

# Usage

```lua
vim.opt.background = "dark" -- or "light"
vim.cmd.colorscheme("sakura");
```
